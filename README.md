# Rocket Engine Test Bench PCB

This PCB was designed as part of my work within IPL (Innovative Propulsion Laboratory), a student association focused on liquid rocket engine development and testing.

## Overview
This board is used within a rocket engine test bench for control and instrumentation purposes. It interfaces with sensors and actuators and is integrated into a larger embedded control system.

## My Contribution
- PCB design (Altium Designer)
- Integration with embedded control systems
- Support for sensor acquisition and actuator control

## Context
This project was developed in a real test bench environment, with practical constraints related to hardware integration, reliability, and iterative testing.
